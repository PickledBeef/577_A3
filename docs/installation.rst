============
Installation
============


Stable release
--------------

To install a3, run this command in your terminal:

.. code-block:: console

    $ pip install ds_a3

This is the preferred method to install a3, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for a3 can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/PickledBeef/ds_a3

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/PickledBeef/ds_a3/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/PickledBeef/ds_a3
.. _tarball: https://github.com/PickledBeef/ds_a3/tarball/master
